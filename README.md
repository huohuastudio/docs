# Spark Studio Docs

Spark Studio 产品文档中心，基于 [VitePress](https://vitepress.dev) 构建。

在线访问：[docs.huohuastudio.com](https://docs.huohuastudio.com)

## 包含产品

- **Novaix** — 一站式 IDC 管理系统
- **Lsky Pro+** — 云端相册系统

## 本地开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```

## 部署

推送到 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。
