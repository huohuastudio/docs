# 实名认证 {#kyc}

Novaix 支持用户实名认证（KYC），管理员可以要求用户在购买实例前完成身份核验。实名认证采用插件化设计，支持多种核验渠道。

## 支持的渠道 {#channels}

| 渠道 | 说明 |
|------|------|
| 阿里云身份核验 | 通过阿里云实名认证 API 核验身份证信息 |
| 腾讯云身份核验 | 通过腾讯云人脸核身 API 核验身份证信息 |

## 配置 {#configuration}

在管理面板的「系统设置」→「实名认证」中选择并配置核验渠道。每个渠道的配置项会以动态表单展示，填写对应的 API 密钥信息即可。

## 认证流程 {#verification-flow}

1. 用户在用户面板的「个人资料」中提交实名认证，填写真实姓名和身份证号
2. 系统调用配置的核验渠道进行身份核验
3. 核验通过后，用户的认证状态会更新为「已认证」

## 管理 {#management}

管理员可以在「用户管理」中查看每个用户的实名认证状态。如果您希望强制要求用户实名认证才能购买实例，可以在「系统设置」中开启实名认证要求。

::: warning
实名认证服务由第三方云平台提供，每次核验会产生费用。请在对应云平台的控制台中查看计费详情。
:::

::: tip
实名认证不是必须的。如果您的业务不要求用户实名，可以不配置此功能，用户将可以直接购买实例。
:::
