# 邮件模板 {#email-template}

Novaix 的各类通知邮件（验证码、订单通知、到期提醒等）都使用预定义的邮件模板。管理员可以自定义这些模板的内容，调整措辞、格式和品牌形象。

## 管理模板 {#manage}

在管理面板的「系统设置」→「邮件设置」中进入邮件模板管理页面。页面会列出所有可用的邮件模板类型：

| 模板类型 | 用途 |
|----------|------|
| 验证码 | 发送邮箱验证码（注册、密码重置、绑定邮箱等） |
| 实例到期提醒 | 实例即将到期的续费提醒 |
| 实例已冻结 | 实例因到期被冻结的通知 |
| 实例已删除 | 实例因过期被自动删除的通知 |
| 流量告警 | 实例流量使用量达到阈值的预警 |
| 实例已限速 | 实例因流量超限被限速的通知 |
| 流量超额计费 | 超出套餐流量后按量计费的扣费通知 |
| 流量超限暂停 | 实例因流量超限被暂停的通知 |
| 自动续费成功 | 自动续费扣款成功通知 |
| 自动续费失败 | 自动续费余额不足通知 |
| 按小时计费失败 | 按小时计费余额不足通知 |
| 工单回复 | 工单有新回复时通知对方 |
| 佣金到账 | 代理商获得新佣金的通知 |
| 退款已通过 | 用户退款申请审批通过通知 |
| 退款已驳回 | 用户退款申请被驳回通知 |
| 余额变动 | 管理员调整用户余额的通知 |
| 节点告警 | 节点资源使用超阈值的告警通知 |

每种模板都有一个系统内置的默认版本。如果您没有自定义某个模板，系统会使用默认版本发送邮件。

## 自定义模板 {#customize}

点击模板列表中的「编辑」按钮，进入模板编辑页面：

- **邮件主题**：邮件的标题行，支持使用变量
- **邮件正文**：HTML 格式的邮件内容，支持使用变量

### 模板变量 {#variables}

模板中可以使用 `{{变量名}}` 格式插入动态内容。编辑页面会列出当前模板可用的所有变量，点击即可复制。常见的变量包括：

| 变量 | 说明 |
|------|------|
| `{{username}}` | 用户名 |
| `{{email}}` | 用户邮箱 |
| `{{site_name}}` | 站点名称 |
| `{{site_url}}` | 站点地址 |
| `{{instance_name}}` | 实例名称 |
| `{{expire_date}}` | 到期时间 |
| `{{balance}}` | 用户余额 |

不同类型的模板可用的变量不同，以编辑页面上列出的为准。

## 预览和测试 {#preview-test}

- **预览**：点击「预览」按钮可以查看模板渲染后的效果，变量会被替换为示例值
- **测试发送**：输入一个测试邮箱地址，发送一封测试邮件，验证模板内容和邮件服务是否正常

## 重置为默认 {#reset}

如果您对自定义的模板不满意，可以点击「重置为默认」恢复到系统内置的模板内容。

::: tip
自定义邮件模板时，建议保留必要的变量（如验证码、链接等），否则邮件中会缺少关键信息。如果您不确定某个变量的作用，可以先通过「测试发送」查看效果。
:::
