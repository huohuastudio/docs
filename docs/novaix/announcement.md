# 公告管理 {#announcement}

Novaix 支持发布平台公告，用于向所有用户传达重要信息，如系统维护通知、新功能上线、促销活动等。公告功能已整合到 [CMS 内容管理](./cms)的文章模块中，使用 `announcement` 文章类型。

## 发布公告 {#publish}

在管理面板的「内容管理」→「文章管理」中，点击「发布文章」按钮，选择类型为「公告」，填写以下信息：

| 字段 | 说明 |
|------|------|
| 标题 | 公告标题 |
| 内容 | 公告正文，支持富文本编辑（格式化文字、链接、图片等） |
| 状态 | 草稿或已发布 |
| 排序 | 排序权重，数字越小排越前 |

## 展示方式 {#display}

公告发布后：

- **用户面板仪表盘**：首页会展示最新的公告条目，用户点击可跳转到公共文章列表页查看详情
- **公共文章列表**：通过 `/articles?type=announcement` 访问，展示所有已发布的公告

## 管理公告 {#manage}

管理员可以在「文章管理」中随时编辑已发布的公告内容或删除不再需要的公告。

::: tip
建议在进行系统维护前至少提前一天发布维护公告，让用户有时间做好准备。对于促销活动类公告，可以在活动结束后及时删除或编辑，避免用户看到过期信息。
:::
