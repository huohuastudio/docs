# 公告管理

Novaix 支持发布平台公告，用于向所有用户传达重要信息，如系统维护通知、新功能上线、促销活动等。

## 发布公告 {#publish}

在管理面板的「公告管理」中，点击「发布公告」按钮，填写以下信息：

| 字段 | 说明 |
|------|------|
| 标题 | 公告标题，展示在公告列表和详情页 |
| 内容 | 公告正文，支持富文本编辑（格式化文字、链接、图片等） |
| 置顶 | 是否将公告固定在列表顶部 |

## 展示方式 {#display}

公告发布后，用户可以在用户面板的「公告」页面查看所有公告。最新的公告排在最前面，置顶公告始终显示在列表顶部。

## 管理公告 {#manage}

管理员可以随时编辑已发布的公告内容或删除不再需要的公告。

::: tip
建议在进行系统维护前至少提前一天发布维护公告，让用户有时间做好准备。对于促销活动类公告，可以在活动结束后及时删除或编辑，避免用户看到过期信息。
:::
