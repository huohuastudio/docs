# ISO 管理

ISO 管理功能允许管理员上传 ISO 镜像文件（如操作系统安装盘、工具光盘等），用户可以将 ISO 挂载到实例的虚拟光驱中使用。

## 管理 ISO {#manage}

### 上传 ISO {#upload}

在管理面板的「ISO 管理」页面，点击「上传 ISO」：

1. 填写 ISO 的名称和描述信息
2. 选择 ISO 文件并上传
3. 上传完成后，系统会自动验证文件并标记为可用状态

ISO 文件存储在 Novaix 工作目录下的 `iso/` 文件夹中。如果配置了[对象存储](./storage)，ISO 文件也会自动备份到远程存储，当本地文件意外丢失时可以自动恢复。

### 删除 ISO {#delete}

在 ISO 列表中点击「删除」，会同时删除数据库记录和磁盘上的文件。

::: warning
删除 ISO 前请确保没有实例正在使用该 ISO。如果有实例挂载了该 ISO，删除不会影响已挂载的实例，但该 ISO 将无法再被其他实例挂载。
:::

## 用户挂载和卸载 {#mount-unmount}

用户可以在实例详情页中管理 ISO 挂载：

### 挂载 ISO {#mount}

点击实例详情页中的「挂载 ISO」按钮，从可用 ISO 列表中选择需要的 ISO 文件。挂载操作是异步执行的，系统会创建一个任务来完成 ISO 的挂载，您可以在任务列表中查看进度。

挂载完成后，实例内部会出现一个虚拟光驱设备，您可以像使用物理光盘一样访问 ISO 中的内容。

### 卸载 ISO {#unmount}

在实例详情页中点击「卸载 ISO」即可。同样是异步操作，完成后虚拟光驱设备会被移除。

::: tip
常见用途：
- 使用 ISO 安装特定版本的操作系统
- 挂载驱动光盘安装驱动程序
- 使用救援工具盘修复系统
:::
